# Harish Satheesh — Portfolio

A single-page, responsive portfolio website for **Harish Satheesh**, AI/ML Engineer, showcasing skills, projects, internship experience, publications, certifications, and education.

🔗 **Live Site:** _add your deployed URL here_

## ✨ Features

- Animated neural-network canvas background
- Terminal-style typing intro effect
- Responsive layout — optimized for desktop, laptop, tablet, and mobile
- Scroll-based section reveal animations
- Filterable project gallery (by category)
- Interactive ML pipeline diagram
- Fully accessible nav with mobile hamburger menu
- Zero build tools — pure HTML, CSS, and vanilla JavaScript

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, Grid, Flexbox, media queries)
- Vanilla JavaScript (IntersectionObserver, Canvas API)
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk), [Inter](https://fonts.google.com/specimen/Inter), and [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts

## 🚀 Running Locally

No build step required.

```bash
git clone https://github.com/Harish180830/<repo-name>.git
cd <repo-name>
open index.html   # or just double-click the file
```

For live-reload during edits, you can optionally serve it with any static server, e.g.:

```bash
npx serve .
```
